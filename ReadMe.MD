# Neo Quotation
This is a short description of a Quotation for a cinema using the semaphore function to prevent overbooking.


## Installation

To install the project, follow these steps:

-  Clone the repository: `git clone https://github.com/Azutech/Grad-Cali`
-  Change to the project directory: `cd project`
-  Install the dependencies: `npm install`
-  Run the project: `npm start`
